## Amplify install

React/Material Ui examples

### aws cli
npm install -g @aws-amplify/cli


### aws react
npm install aws-amplify @aws-amplify/ui-react




### aws js config
```javascript

import Amplify from "aws-amplify";
import awsExports from "./aws-exports";
Amplify.configure(awsExports);

```
